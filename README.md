<h1>🌍 Global Blood Pressure Insights Dashboard (Excel)</h1>

<h2>📌 Description</h2>
<p>
This project presents an <b>interactive Excel dashboard</b> designed to analyze global health data related to 
<b>blood pressure, BMI, lifestyle factors, and patient demographics</b>.
</p>
<p>
The dashboard provides insights into <b>hypertension trends, smoking habits, alcohol consumption, and health risks</b> 
across different countries, age groups, and genders, enabling effective data-driven analysis using Microsoft Excel.
</p>

<h2>🚀 Key Features</h2>

<h3>🔹 Health Metrics Overview</h3>
<ul>
<li>Average BMI tracking</li>
<li>Average Systolic & Diastolic Blood Pressure</li>
<li>Blood Pressure category distribution</li>
</ul>

<h3>🔹 Demographic Analysis</h3>
<ul>
<li>Patients distribution by country</li>
<li>Gender-based analysis</li>
<li>Age group-wise blood pressure comparison</li>
</ul>

<h3>🔹 Lifestyle Impact Analysis</h3>
<ul>
<li>Smoking status impact on health</li>
<li>Alcohol consumption vs hypertension trends</li>
<li>Diabetic vs non-diabetic population analysis</li>
</ul>

<h3>🔹 BMI & Health Risk Insights</h3>
<ul>
<li>BMI category distribution (Normal, Overweight, Obese, Underweight)</li>
<li>Relationship between BMI and blood pressure</li>
</ul>

<h3>🔹 Interactive Filters</h3>
<ul>
<li>Year-wise filtering</li>
<li>Country selection</li>
<li>Gender-based filtering</li>
</ul>

<h2>📊 Dashboard Highlights</h2>
<ul>
<li>Clean healthcare-themed UI design</li>
<li>Interactive charts and KPIs using Excel</li>
<li>Dynamic filtering with slicers</li>
<li>Multi-dimensional data analysis</li>
</ul>

<h2>🛠️ Tools & Technologies Used</h2>
<ul>
<li>Microsoft Excel – Dashboard creation and visualization</li>
<li>Pivot Tables & Pivot Charts</li>
<li>Slicers & Data Filtering</li>
<li>Formulas & Data Analysis</li>
</ul>
